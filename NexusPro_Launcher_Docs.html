üîß NexusProIniConverter.exe

üìñ Overview & Purpose

The NexusProIniConverter.exe is a security utility that converts plain-text INI configuration files into encrypted binary format. This tool is essential for protecting sensitive launcher configuration data from unauthorized access or modification.

üéØ Primary Function: Converts NexusPro_Launcher.ini files into encrypted NexusPro_Launcher.bin files using AES-256 encryption.

üöÄ How to Use

For End Users (Simple Method)

  1. Prepare Files: Copy your NexusPro_Launcher.ini file to the same folder as NexusProIniConverter.exe
  2. Run Converter: Double-click NexusProIniConverter.exe to start the conversion process
  3. Wait for Completion: The tool will automatically create NexusPro_Launcher.bin in the same folder
  4. Deploy: Copy the generated .bin file to your launcher's configuration directory
  5. Cleanup: The original .ini file can be safely deleted after verification

For Developers (Command Line)

cd NexusProIniConverter dotnet build dotnet run # or .\NexusProIniConverter.exe

üîê Licensing System

The converter includes an integrated licensing system that validates usage before allowing conversion operations.

  2. Generate License: Use NexusProLicenseGenerator.exe to create a license file
  3. Install License: Place the generated nexusProLic.bin file in the NproLic folder
  4. Verify Installation: Run the converter to validate the license

üîÑ Auto-Update System

How Auto-Update Works

  1. Server Scanning: Automatically detects all files and folders on the update server
  2. File Comparison: Compares local files with server versions using cache and timestamps
  3. Selective Download: Downloads only files that are new or have been updated
  4. Progress Tracking: Shows individual file progress and overall update progress
  5. Error Handling: Continues with remaining files if individual downloads fail

Configuration Example

[Updater] ; Auto-updater configuration UpdateUrl=https://your-server.com/updates/ Enabled=true

Server Requirements

  • Directory Listings: Server must provide directory listings (Apache with Options +Indexes)
  • HTTP/HTTPS Access: Allow web access to update files
  • Folder Structure: Maintain same folder structure as local installation
  • Security: Use HTTPS for secure file transfers
‚ö†Ô∏è Important: The auto-updater will overwrite existing files without creating backups. Ensure your update server contains only verified, safe files.
